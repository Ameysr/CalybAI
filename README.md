# CalybAI — Research Paper Onboarding

Maps a research field's citation network to generate reading curricula.
Helps new researchers know what to read first and in what order.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python src/main.py --topic "Retrieval Augmented Generation"
```

Outputs a reading curriculum with foundational papers, clusters, and
dependency-respecting reading order.
